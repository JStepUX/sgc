# Changelog — May 2026

- **Project bootstrap.** Initialized the SGC repository around the Phase 1.5
  artifact (`sgc-phase-1-5.jsx`). Established the developer-experience scaffold,
  adapted from the Miles-Chat project: git repo, `.gitignore`, `CLAUDE.md`,

  (`project-coordinator`, `task-executor`, `scout`), two skills (`pre-commit-qa`,
  `bash-tools`), and the `pre-commit-gate.mjs` PreToolUse hook that gates
  `git commit` behind a fresh QA marker.
- **Agent scripts.** Added `scripts/agent/` with `codebase-snapshot.sh`,
  `git-context.sh`, `related-files.sh`, `health-check.sh`, and shared
  `_common.sh`. Trimmed from the Miles-Chat set to fit a single-file React
  project with no build tooling.
